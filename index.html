Cách kết nối Notion
1
Vào notion.so/my-integrations → tạo Integration mới → copy Internal Integration Token
2
Mở database Activity Log trong Notion → Share → Invite integration bạn vừa tạo
3
Copy Database ID từ URL: notion.so/[workspace]/DATABASE_ID?v=...
4
Nhấn ⚙️ bên dưới → paste Token & Database ID → Kết nối
5
Tạo database với properties: Date (date), Type (select), Status (select), Intensity (number)
Nhấn ⚙️ để kết nối Notion, hoặc xem Demo để trải nghiệm trước
